# ewwvolume

volume control widget for eww

## installation

1. install the binary:
```
   cargo install --path .
```

2. run the install script:
```
   bash install.sh
```

3. add this to your eww.yuck:
```
   (include "./ewwvolume/ewwvolume.yuck")
```

4. paste the contents of ewwvolume.css into your eww.css
